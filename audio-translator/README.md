# 实时音频翻译字幕 Chrome 插件

一款 Chrome 浏览器插件，能够实时捕获当前标签页的音频流，通过 AI 进行印尼语到中文的实时翻译，并以悬浮字幕形式展示在网页上。

## 功能特性

- **实时音频捕获**: 使用 Chrome tabCapture API 捕获浏览器标签页音频
- **AI 实时翻译**: 支持 OpenAI Whisper 语音识别 + GPT 翻译
- **悬浮字幕显示**: 半透明可拖动字幕窗口，实时显示翻译结果
- **多语言支持**: 支持印尼语、英语、日语等多种语言翻译
- **快捷键支持**: Alt+T 开始/停止翻译，Alt+H 显示/隐藏字幕
- **鲁棒性设计**: 页面刷新自动恢复，支持非标准视频播放器

## 安装方法

### 开发者模式安装

1. 下载或克隆本项目到本地
2. 打开 Chrome 浏览器，进入 `chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择本项目文件夹 `audio-translator`
6. 插件图标将出现在 Chrome 工具栏

## 使用方法

### 首次使用

1. 点击 Chrome 工具栏的插件图标
2. 点击"设置"进入配置页面
3. 填写 OpenAI API Key（必填）
4. 选择源语言和目标语言
5. 调整字幕样式（可选）
6. 保存设置

### 日常使用

1. 打开包含印尼语视频的网页
2. 点击插件图标，选择"开始翻译"
3. 字幕悬浮窗将自动出现在网页上
4. 可以拖动字幕窗到任意位置
5. 可以滚动查看历史字幕
6. 点击"停止翻译"结束捕获

### 快捷键

- `Alt + T`: 快速开始/停止翻译
- `Alt + H`: 显示/隐藏字幕窗口

## 配置说明

### API 配置

- **提供商**: 支持 OpenAI 和豆包大模型
- **API Key**: 您的 API Key
  - OpenAI: 访问 [OpenAI Platform](https://platform.openai.com/)
  - 豆包: 访问 [火山引擎](https://console.volcengine.com/)
  - 安全性: API Key 仅存储在本地，不会上传

#### 豆包大模型配置

1. 访问 [火山引擎控制台](https://console.volcengine.com/)
2. 开通方舟大模型服务
3. 创建推理接入点，获取模型 ID（以 `ep-` 开头的端点 ID）
4. 创建 API Key
5. 在插件设置中选择"豆包大模型"，填写：
   - API Key: 火山引擎 API Key
   - 模型 ID: 推理接入点 ID（如 `ep-20240101-xxxxx`）
   - 端点地址: 默认 `https://ark.cn-beijing.volces.com/api/v3`

### 翻译设置

- **源语言**: 音频源语言（默认印尼语）
- **目标语言**: 翻译目标语言（默认中文）

### 字幕样式

- **字体大小**: 14px - 20px
- **最大显示行数**: 3 - 10 行
- **背景透明度**: 30% - 100%
- **自动滚动**: 新字幕自动滚动到最新

## 技术架构

```
chrome-audio-translator/
├── manifest.json          # 扩展配置文件
├── background.js          # Service Worker 后台脚本
├── content.js             # 内容脚本，注入字幕 UI
├── content.css            # 字幕样式
├── popup.html/js/css      # 弹出页
├── options.html/js/css    # 设置页
└── icons/                 # 图标文件
```

### 核心组件

- **Background Script**: 管理音频捕获、AI 通信、状态管理
- **Content Script**: 注入字幕悬浮窗、处理用户交互
- **Popup**: 快速控制面板
- **Options**: 详细设置页面

### 数据流

1. 用户点击"开始翻译"
2. Background 调用 `chrome.tabCapture.capture()` 获取音频流
3. MediaRecorder 每 5 秒录制音频片段
4. 音频发送到 OpenAI Whisper 进行语音识别
5. 识别结果发送到 GPT 进行翻译
6. 翻译结果通过消息传递到 Content Script
7. Content Script 更新字幕显示

## 注意事项

1. **API 费用**: 使用 OpenAI API 会产生费用，请注意用量
2. **隐私安全**: 音频数据会发送到 OpenAI 服务器进行处理
3. **网络要求**: 需要稳定的网络连接以保证实时翻译
4. **浏览器支持**: 仅支持 Chrome 浏览器（Manifest V3）

## 故障排除

### 无法捕获音频

- 确保页面正在播放音频
- 检查是否已授予音频捕获权限
- 尝试刷新页面后重试

### 翻译不工作

- 检查 API Key 是否正确配置
- 检查网络连接是否正常
- 查看控制台错误信息

### 字幕不显示

- 检查是否点击了"显示/隐藏字幕"
- 刷新页面后重新启动翻译
- 检查是否有其他扩展冲突
